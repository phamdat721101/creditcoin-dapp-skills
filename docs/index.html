What It Does

Everything you need to
build on Creditcoin.

The skill covers the full development lifecycle from Solidity contracts to frontend integration.

Solidity Contracts

Write EVM-compatible smart contracts in Solidity. Deploy with Hardhat or Foundry to the Creditcoin EVM environment.

Ethers.js / Web3.js

Interact with the Creditcoin Network using standard Ethereum libraries. Manage wallets and submit transactions effortlessly.

MetaMask Integration

Seamlessly connect to MetaMask for managing CTC tokens and user signing.

Credit History

Automatic verification and recording of credit history on the Creditcoin ledger.

Testing & Simulation

Test your Solidity contracts locally using Hardhat Network. Simulate transactions and verify logic.

Real World Assets

Build applications that tokenize and manage Real World Assets (RWA) with transparent credit history.

USC Cross-Chain

Build trustless multichain dApps using Universal Smart Contracts with STARK proof verification.

Quick Start

Install the skill,
start building.

One command installs everything. Claude Code automatically loads the skill and its reference docs when you ask about Creditcoin development.
